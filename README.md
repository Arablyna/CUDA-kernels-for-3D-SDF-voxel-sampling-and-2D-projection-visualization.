# CUDA-kernels-for-3D-SDF-voxel-sampling-and-2D-projection-visualization.
CUDA/C++ project for 3D signed distance field visualization. It samples a torus SDF on a voxel grid using GPU kernels, maps 3D data to contiguous memory, and projects the volume into 2D images. Demonstrates CUDA indexing, block/grid configuration, and basic GPU visualization.
